# DIAMONDZ

안녕하세요.

팀 프로젝트로 주얼리 쇼핑몰을 만들었습니다. 상품명, 카테고리로 검색이 가능합니다.

각 카테고리별로 상품을 쉽게 볼 수 있는 UI를 구축했습니다. 많은 이용 부탁드립니다
